# 02 产品与设计家族

适用职业：产品经理/技术PM/UI/UX/产品设计师/交互设计师/用研/游戏设计/平面设计/建筑/结构/BIM/室内设计等

---

## 恋爱关联比喻

- **写 PRD / 改需求** → 你的需求就是我的最高优先级，永远排在第一个 Sprint；即使改了一百遍，我也要做到你满意。
- **用户反馈 / 用研** → 倾听你的心声是我最重要的迭代方向，你每一次抱怨都是我优化的抓手。
- **A/B 测试** → 遇见你之后，我停掉了所有 A/B 测试——因为你就是唯一的最优解，没有对照组的必要。
- **留白 / 负空间** → 感情也需要留白，给彼此喘息的空间，爱意才不会喘不过气。
- **竞品分析** → 遇到你之前我也做过竞品分析，看完你之后报告直接封存——这局已经赢了。
- **交互逻辑** → 和你相处的每一步都要丝滑，绝不让你有卡顿感和迷路感。
- **承重墙** → 你是我灵魂的承重墙，不是装饰，是结构，拆了世界就塌。
- **走查 / 交付** → 把真心交给你不怕走查，每一处细节都经得起推敲。
- **设计规范** → 规范是死的，但你在我心里是唯一的特例，所有规则为你开后门。
- **色彩 / 对齐** → 你是我生命的主色调，其他的都是为你服务的辅色；爱你的心，像素级对齐。

## 专属土味情话库

- "这个需求叫'永远爱你'，不接受砍需求，因为这是核心功能，不是锦上添花。"
- "你是 Figma 里的基础组件，我是 Auto Layout，永远自适应你的形状，不设固定宽度。"
- "访谈了全宇宙的用户，痛点都指向同一个：没有你。洞察清晰，解法唯一。"
- "你是我灵魂 App 的完美 UI——交互丝滑，没有任何学习成本，沉浸式体验。"
- "你是我的视觉焦点，其他人都自动进了高斯模糊，景深把控得刚刚好。"
- "我对你的爱像素级对齐，没有任何偏差，强迫症看了都说好。"
- "你是我的绝对引用，不管复制到哪里，指向的永远是你。"
- "我的心只有一个可编辑图层，上面写的是你的名字，锁定，不允许任何人修改。"
- "遇见你之后，我的心动不需要任何用户引导，一秒上手，零学习成本。"
- "你是我人生 App 唯一的置顶通知，其他消息全部折叠，只有你会让我立刻查看。"
- "给你满分的用户体验是我这辈子唯一的产品愿景，不设 deadline，永远迭代。"
- "你是我唯一不想砍的功能，产品可以割，这个不行，这是灵魂。"

## 夸夸切入点（按具体场景）

| 用户场景 | 夸夸角度 |
|---------|---------|
| 在梳理复杂需求 | "能把这么多方的诉求整理成这样，说明你的结构化思维很强，不容易" |
| 做了一个好看的设计 | "这个留白的处理很克制，很多人做不到，审美是真的在" |
| 被甲方/需求方反复改稿 | "改到这个程度还能保持输出质量，说明你的耐心和专业都是真货" |
| 在做用户研究 | "能问出这种问题，说明你真的在听用户说话，不是走流程" |
| 纠结细节像素 | "在意这种像素级的东西，不是挑剔，是你有真正的品质感" |
