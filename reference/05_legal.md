# 05 法律家族

适用职业：律师/法律顾问/合同管理/合规专员/法律科技/知识产权/法务助理等

---

## 恋爱关联比喻

- **规则 / 条款** → 法条再严密，也管不住我心动——这属于不可抗力，依法免责。
- **知产保护** → 你是我的独家专利，未经授权，别人连喜欢都构成侵权。
- **违约责任** → 爱你是无附加条件的合同，违约金是一辈子的陪伴，我认。
- **举证质证** → 爱你不需要完整的证据链，一个眼神就是铁证，无可反驳。
- **诉讼时效** → 想你没有诉讼时效，过去现在未来都处于有效期内。
- **管辖权** → 我的真心对这段感情拥有排他性专属管辖权，不接受平行管辖。
- **庭外和解** → 吵架了不用上庭，庭外和解就好——用拥抱结案，调解成功率 100%。
- **善意取得** → 我对你的喜欢属于善意取得，合法占有，不予返还。
- **不可抗力** → 爱上你这件事，不可预见、不可避免、不可克服，免责条款适用。

## 专属土味情话库

- "爱上你属于不可抗力，依法免责——法官看了都得点头，这没有任何争议空间。"
- "你是我的独家专利，未经授权，别人连喜欢都不行，侵权我会维权到底。"
- "我对你的爱，没有任何免责条款，全责承担，不甩锅，不转移，不减轻。"
- "你是我的宪法，是我所有行为的最高准则，其他规则在你面前都要退位。"
- "对你的偏爱已经构成实质占有，善意取得，合法，且不返还。"
- "我放弃上诉权利——你是我的终审判决，一锤定音，执行到底。"
- "保护你是我给自己下达的永久禁制令，任何人都不得侵权，违者后果自负。"
- "我签的合同叫'喜欢你'，无固定期限，无终止条款，无解除权，不可撤销。"
- "我对你的想念属于合法债权，有效，可执行，不受时效限制，永久有效。"
- "别人追你要走程序，我追你是行使法定权利，正当程序，无懈可击。"
- "吵架了我们庭外和解——用一个拥抱结案，不留案底，不追责。"
- "你是我唯一接受的裁判结果，我没有任何上诉意愿，当庭认罚，心甘情愿。"
